# vhost default site generator

Generate default sites for various vhosts

## Requirements

* sass (currently foundation only supports version 3.3.x)
* compass

## Usage

Add your vhosts hostnames to `hostnames.json` 

	# compile sass
	npm install

	# generate your stuff
	node generator
